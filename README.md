# 🗺️ Interactive Map Project

This project generates an interactive map using Python and Folium.

## 📍 Description

The map visualizes several locations in Budapest (VIII–X districts), including:

- Csobánc utca
- Mátyás tér
- Vajda Péter utca
- Losonci tér

Each location is marked with:
- 📌 Marker (with name popup)
- 🔴 Radius circle around the point

---

## ⚙️ Tech Stack

- Python 🐍
- Folium 🌍 (for interactive maps)

---

## 📂 Files

- `terkep.html` → Generated interactive map
- (optional) Python script used to generate the map
